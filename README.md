# 📡 Sales Intelligence — Futuristic Business Analytics Dashboard

A premium multi-page **Streamlit analytics dashboard** for sales intelligence, profitability tracking, customer analytics, product intelligence, discount optimization, and business performance monitoring with a futuristic cyber-themed UI.

---

## 🚀 Features

* 📊 Real-Time Sales Intelligence
* 💰 Profit & Margin Analysis
* 🛍️ Product Intelligence Dashboard
* 🎯 Discount Optimization Analysis
* 👥 Customer Segmentation & Retention Tracking
* 🔄 Return & Damage Monitoring
* 🏆 Sales Representative Performance Analytics
* 📈 Interactive KPI Dashboards
* 🎨 Futuristic Glassmorphism UI
* 🔍 Dynamic Sidebar Filters
* ⚡ Fast Cached Data Processing
* 📉 Advanced Plotly Visualizations

---

## 🖥️ Dashboard Pages

| Page                      | Description                                         |
| ------------------------- | --------------------------------------------------- |
| 📡 System Overview        | High-level business performance summary             |
| 💸 Sales Analysis         | Geographic, channel & temporal sales breakdown      |
| 📈 Profit Analysis        | Margin analysis & profitability insights            |
| 👥 Customer Analysis      | Segmentation, retention & customer behaviour        |
| 🔄 Return Analysis        | Return trends, damage tracking & risk signals       |
| 🏆 Sales Rep Intelligence | Leaderboards, efficiency & rep performance          |
| 🛍️ Product Intelligence  | Product trends, category insights & performance     |
| 🎯 Discount Analysis      | Discount impact, profitability & sales optimization |

---

## 🛠️ Tech Stack

* **Frontend:** Streamlit
* **Visualization:** Plotly Express & Plotly Graph Objects
* **Data Processing:** Pandas & NumPy
* **Styling:** Custom Futuristic CSS
* **Architecture:** Multi-page Streamlit Application
